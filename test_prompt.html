git 사용 팁

    깃 지역 저장소 설정 [git init]
    사용자 등록 [git config user.name "사용자 이름"]
    사용자 등록 [git config user.email "이메일 주소"]
    깃 원격 저장소 설정 [git remote add origin {복사한 원격 저장소 주소}]
    프로젝트의 현재 파일 상태를 확인 [git status]
    커밋에 포함될 파일로 등록 [git add "파일명 or 폴더이름"] / [git add .](현재 디렉토리의 전체 파일을 등록)
    커밋 [git commit](커밋 내역 상세 작성), [git commit -m](커밋 메세지 단순 작성)
    프로젝트 작업 이력 확인 [git log --pretty=oneline --graph]
    저장된 커밋을 원격 저장소에 등록 [git push {저장소} {브랜치}]
    원격 저장소 복제 [git clone "원격 저장소 주소" "새로운 저장소 이름"]
